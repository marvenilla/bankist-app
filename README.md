# Bankist App

The Bankist App is a simulated banking application designed to manage personal banking transactions. This project was created as part of the educational curriculum for "The Complete JavaScript Course 2024: From Zero to Expert!" by Jonas Schmedtmann on Udemy. It features functionalities such as logging in, transferring money, requesting loans, and closing accounts.

## Features

- **User Authentication:** Simple login mechanism to simulate access control.
- **Transaction Management:** Users can transfer money and request loans.
- **Account Operations:** Users can view their current balance and past transactions, and close their accounts.

## Technologies Used

- **HTML** - Structure of the web pages.
- **CSS** - Styling of the application.
- **JavaScript** - Functionality and event handling.

## Setup

To run this project, download it locally, extract the files, and open `index.html` in your web browser.
